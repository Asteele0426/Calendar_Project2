## Event Calendar
  
A calendar application that allows the user to view a list of events and their details, add events, and add guests to a guest list. 

## Technologies Used
    -HTML
    -CSS
    -Javascript
    -Node
    -Express
    -MongoDb
    -Mongoose
    -OAuth
